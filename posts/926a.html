4.三和弦/七和弦

阿豪2023-02-272025-04-03
文章内容： 和弦篇[三和弦/七和弦]

和弦是什么?

弦定义：叠置而成的一种和弦

三和弦：

  • 以三度关系叠加的两组音(至少是一个两组音去叠加的三度关系)，就叫三和弦

  • 根音到三音是一个三度关系，三音到五音又是一个三度关系

  • 三和弦有三个音构成：根音 - 三音 - 五音 (冠音) 关系为三度叠加

概念: 以三度关系叠加出来的三个音，就是一个三和弦

  • 三和弦的性质：大三、小三、增三、减三 排列出来这四种特性的和弦

[大三度 + 小三度]

大三和弦的理解方式

  • 根音到三音是大三度，三音到五音是小三度，根音到五音是纯五度的关系。

  • 大三和弦一般会写成Major/Xmaj&Xm 或者C和弦直接写成C,D和弦直接写成D指的就是大三和弦

[小三度 + 大三度]

小三和弦的理解方式

  • 小三和弦的排列和大三和弦相反

  • 排列方式：根音 - 降三音(小三) 降三音 - 五音(大三)，同样也是纯五度的关系

  • 降三音的作用：把原本的大三度变成小三度

  • 小三和弦标记符号：Minor/Xmin Xm

[小三度 + 小三度]

  • 和小三和弦的区别在于根音到五音不是纯五度了，而是增四减五度

  • 排列：根音 - 降三音(小三) 降三音 - 降五音(小三)

  • 减三和弦标记符号：XdimXm

平常用到的不多，最常见的就是大三、小三，有时候增三和弦比减三和弦用的都多

[大三度 + 大三度]

  • 排列：根音 - 三音 三音 - 升五音，就是一个增三和弦

  • 增三和弦标记符号：Xaug

简单的理解

  • 大三和弦

    • 大三 + 小三

  • 小三和弦

    • 小三 + 大三

  • 减三和弦

    • 小三 + 小三

  • 增三和弦

    • 大三 + 大三

以上为三和弦的基础框架理论

七和弦：

七和弦的特性排列组合方法

由四个音按三度叠置而成的一种和弦

  • 三和弦是三度加三度， 七和弦就是再往上叠加一个三度，就会产生多一个七音

  • 三个三度关系去叠加

  • 排列：根音 - 三音 - 五音 - 七音

  • 一般会写为：Major7

  • 以大三和弦为基础，根音与七音相距为大七度的和弦

  • 排列：根音 - 三音(大三度) 三音 - 五音(小三度) 五音 - 七音(大三)

  • 大小七和弦：以大三和弦为基础，根音与七音相距为小七度的和弦

  • 排列：根音 - 三音(大三度) 三音 - 五音(小三度) 五音 - 降七音(小三)减三和弦

  • 以小三和弦为基础，根音与七音相距为小七度的和弦

  • 排列：根音 - 降三音(小三度) 降三音 - 五音(大三度) 五音 - 降七音(小三)

  • 以减三和弦为基础，根音与七音相距为小七度的和弦

  • 包含增四减五度，又包含了减三和弦的色彩

  • 半减七和弦：m7-5

  • 排列：根音 - 降三音 降三音 - 降五音 降五音 - 降七音

  • 以减三和弦为基础，根音与七音相距为减七度的和弦

  • 排列：根 降三 降五 重降七

  • 以小三和弦为基础，根音与七音相距为大七度的和弦

  • 排列：根 降三 五 七
