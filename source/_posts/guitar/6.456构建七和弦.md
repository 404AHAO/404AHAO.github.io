# 认识指板上所有的七和弦

- 跟`三和弦`一样，也是在四、五、六弦的不同`八度`方向去构建所有的七和弦
- `一三`指型：原位指型
- `一五`指型：Drop和弦(七和弦的Drop和弦就是`一五`指型)

# 四弦

{% note info modern %}一三指型{% endnote %}
```
大七：根音 三音 五音 七音 | 七音是一弦的空弦
小七：根音 降三音 降七音 根音 | 会省略了五音
小大七：根音 降三音 七音 根音 | 省略了五音
大小七：根音 三音 降七音 根音 | 省略了五音
```
![](https://image.ahao.ah.cn/2140079466d5a0f8b077cf8c530c5ae.png)

{% note info modern %}一五指型{% endnote %}

```
大七：根音 五音 七音 三音 | 随着根音的变化，和弦的名称、姓氏也会发生变化
小七：根音 五音 降七音 降三音
小大七：根音 五音 大七音 降三音 | 有增三和弦的色彩，一般在二级和弦才会用到
大小七：根音 五音 降七音 三音
半减七：根音 降五音 降七音 降三音
减七：根音 降五音 重降七 降三音
```

# 五弦
{% note info modern %}一三指型{% endnote %}

```
大七：根音 三音 五音 七音 三音 | 最后的三音在一弦的位置
小七：根音 降三音 降七音 降三音 | 省略五音
小大七：根音 降三音 七音 降三音
大小七：根音 三音 降七音 根音 | 省略五音
```
![](https://image.ahao.ah.cn/2166d5761fb4b6ffb2e6aa764475dba.png)

{% note info modern %}一五指型{% endnote %}

```
大七：根音 五音 七音 三音
小七：根音 五音 降七音 降三音
小大七：根音 五音 降七音 降三音
大小七：根音 五音 降七音 三音
半减七：根音 降五音 降七音 降三音
减七：根音 降五音 重降七 降三音
```

# 六弦

{% note info modern %}一三指型{% endnote %}

```
大七：根音 三音 七音 根音 五音 | [指型只能在G和弦使用]
小七：根音 降七音 降三音 五音
大小七：根音 三音 降七音 根音
半减七：根音 降七音 降三音 降五音
减七：根音 降七音 降三音 降五音
```
{% gallery true,,2 %}
![](https://image.ahao.ah.cn/d00470f20f1c6ba95d7201576c6f36c.png)
![](https://image.ahao.ah.cn/534da0ee8fde2694811c8de7a963b3f.png)

{% note info modern %}一五指型{% endnote %}

```
大七：根音 七音 三音 五音
小七：根音 五音 根音 降三音 降七音 根音
小大七：根音 七音 降三音 五音
大小七：根音 五音 降七音 三音 五音 根音
半减七：根音 降五音 降七音 降三音 降五音 | 降五音可以省略掉，跟一三指型一样即可
减七：根音 降五音 重降七 降三音 降五音
```
