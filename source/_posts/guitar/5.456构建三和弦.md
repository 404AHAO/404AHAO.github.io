# 三和弦

弹过吉他的都知道一般情况下吉他上的`根音弦`，就是在它的四弦、五弦和六弦这三根低音弦。

我们将吉他六根琴弦分一下组的话，`456`弦基本上就是低音区。然后二三弦是中音区，一弦或者一二弦属于高声部的区域。

知道这个关系之后，我们的根音都会放在`四弦`、`五弦`、以及`六弦`上。

我们知道这个概念之后，我们可以更轻松的在指板上去学习和弦的构建方法。

先从`四弦`开始，然后再到`五弦`，再到`六弦`，逐渐去梳理一下所有的三和弦。

{% note info simple %}小贴士{% endnote %}

```
>>> 简要

吉他上的根音弦就在它的四、五、六弦三根低音弦

六根弦分组：
- 4、5、6弦为低音区
- 2、3弦为中音区
- 1、2弦为高声部的区域
```

{% tabs 四弦 %}
<!-- tab 四弦 -->

前提要点：

- 吉他是一个指型类乐器，
  - 根音接三音
  - 根音接五音
  - 也称之为：`1315`指型和弦

```

大三和弦的组成：135(根音到三音是大三度，三音到五音是小三度 就叫`大三和弦`)

小三和弦的组成：根音到三音是小三度，三音到五音是大三度，跟反过来的就是`小三和弦`

```

在指板上没办法跟键盘一样的`一三五`去排列，所以有可能会出现`一五一三`的排列

也就是吉他上的`根音`完了再出现`五音`，或者~~1~~`根音`完了出现`三音`

出现三音的这个地方就是上方所提到的`一三的排列`和`一五的排列`

> 一三和一五的排列取决于什么呢？

- 取决于指板上的量个不同的`8度方向`

从四弦开始讲解一下`8度`的方向

*假设：*

- 四弦上的音，先拟定一个音 f这个音在C调当中理解为`fa`，往前或者往后都有一个8度关系

- 有这两个8度关系，我们就有`从前`和`从后有`两种和弦的`排列方法`

从下列图往回的方向去走的方法，它的排列方法为 `根音-->三音` `三音-->五音` `五音-->根音`， 也就是`根三五根音`，正好对应着`一三五一`的排列

- 它是一个小横按的`f`和弦

![](https://image.ahao.ah.cn/859e5cb3972c20eddf2738cd0d37a13.png)

和弦往后走的话其实是一个五音，排列方法为 `根音-->五音` `根音-->三音` 也就是`根三根五`, 对应着`一五一三`的排列方式

- 如果往前去找八度的话，它是一个`d`和弦的指型

![](https://image.ahao.ah.cn/71f1bd5ab871a7af71c99c1a5bfbeee.png)

- 这就是四弦上的两个指型

虽然以上两种排列的方式不一样，但它们的组成音都是`135`组成的，这个就是吉他指板上比较奇妙的点

因为它有不同的`八度`方向，所以导致了它的`排列方式`会不一样

<!-- endtab -->

<!-- tab 五弦 -->

五弦上的大三和弦也是相同的方法

我们所熟悉的C和弦也是相同的排列方式，从c开始

- 依旧是 根音-->三音 五音-->根音

  - 往回去推，它是一个`C`和弦的指型

![](https://image.ahao.ah.cn/8cfe6cb79d138a07dc37dede59c29b6.png)

- 根音-->五音 根音-->三音 排列

- 往前去推一个8度的根音，它是一个`A`和弦的指型

![](https://image.ahao.ah.cn/26b9aca11c3a5b04503291091281f9e.png)

<!-- endtab -->

<!-- tab 六弦 -->

我们假设`g`这个音，拟定为根音`1`

- 根音-->三音 五音-->根音 ，它就是G和谐的一个指型，也是有一个八度方向的

  - 三弦的空弦等于`g`,六弦的三品等于`g`,它正好是一个`8度`

![](https://image.ahao.ah.cn/97502d8b5bcbd7d2bb9134b19fa1093.png)

- 根音-->五音 根音-->三音 然后继续 **五音** 到 **根音** 就排列出来了

  - 再往前推，是一个`E`指型，也可以说是一个大横按的`F`

![](https://image.ahao.ah.cn/83ac20894b6584837af56233801131b.png)

<!-- endtab -->

{% endtabs %}

以上就是四五六弦不同八度方向所有的大三和弦排列

> 总结一下这几个指型的三和弦具体是什么指型？

学习吉他最重要的一个系统：

- C、A、G、E、D系统，也叫caged系统(caged是一个名称，CAGED是拆分开来的，是一个道理)

- 每一个456弦上，每一根弦两个八度方向都有一个不同的指型

- 四弦指型：小横按的`f`(大横按的`f`其实也是`e`往后推出来的)和`d`

- 五弦指型：`c`和`a`

- 六弦指型：`g`和`e`

四五六弦对应的逻辑正好是： C、A、G、E、D五个指型或者叫六指型(因为多了一个小横按的`f`指型)
