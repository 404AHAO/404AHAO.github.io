# 关于我的 2022 年度总结

## 2022 年度总结
---------------------------------------------------------------------------------------------------------

因为一直没时间，所以拖到现在赶紧补上，顺便写一下博客，当然 

会跟编程星球的内容有点区别(毕竟白天要睡觉了时间不太够)

兜兜转转现在也一年过去了，感觉无论计划什么事都没有任何进展，干脆不打算做计划了，

从刚开始学 python 到现在。。。(好吧，是我太蔡了~~)，

刚学的时候还幻想着一个月入门三个月接单半年结业，

一年迎娶白富美走上人生癫疯。。。。


> 关于学习
 
```
发现自己学习的基础方面还是挺薄弱的，

争取今年尽量全部补回来，去年立的 flag 好像都一半没有完成，今年就先不立了把去年的目前做完再说

由于现在对于JS逆向比较感兴趣，所以这段时间也还做摸索(恳求各位大佬们手下留情，网站别改太快了)

吉他也好久没拿出来玩了，这段时间恢复了也要开始给自己一点时间练习练习(有好听的歌可以推荐一下，下方可以留言哦)，希望还能唱 跳 rap 🏀，咳咳咳

```
到时候可以和同学还有朋友们把酒欢歌,好吧，由于太宅了，其实也没多少时间(ps:未成年不要喝酒哦~)

> 现在每天的安排：
```
1. 看爬虫逆向跟机器学习的视频回放(基本跟着敲代码，争取早日成为CV大佬)

2. 练习一些逆向的题目网站(还是得多看看相关法规，别哪天真给逆进去了。。) 

3. 继续整理博客，争取多做些笔记目前文章都没几篇，主要大部分都还放在语雀，就算有也是水货，害~(当时花费了一个多月得时间搭建，期间还一直麻烦三水大佬)
```

> 关于脱单的计划(拿来水字数的，其实没有计划)：
```
- 完全没有这个打算

- 现在也变成单身主义者，身边的人结婚的结婚，我还是坚守着要一直单身(其他人可不要学我，这不好)

- 就是曾经答应侄子侄女的额...以后再说(现在面对小孩比面对大人还可怕)
```

期间其实有过很多担忧跟迷茫的，有时突然有目标了就很认真在对待，

也有时候都不知道自己到底想要什么在那里发呆，甚至一段时候差点自闭了。。。

主要还是有一段时间身体出现问题，那一个多月天天跑医院没法顾虑其他(不说这些不好的了。。)


> 最后当然还是今年的愿望啦(再水一下)
```
当然今年也希望能去一趟周董的演唱会，也是这么多年以来的愿望之一(都快成执念了)

今年有机会的话还是要去看下其他朋友们(例如：重庆，深圳等等。。)，好像还挺多

最后就是希望家人能健健康康，平平安安的就满足了
```

博客的内容原本想着做一些资料笔记，但感觉还是作为自己的日记更为适合，哈哈哈哈哈

在新的一年里祝大家学业进步，万事如意，心想事成，多多发财(我也想呜呜呜呜呜~~~)
